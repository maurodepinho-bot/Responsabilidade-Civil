## 🚀 Como usar no GitHub Pages

### 1. Criar Repositório
1. Faça login no GitHub
2. Clique em "New repository"  
3. Nome sugerido: `campos-pinho-site`
4. Marque como "Public"
5. Clique em "Create repository"

### 2. Fazer Upload dos Arquivos
1. Clique em "uploading an existing file"
2. Faça upload de todos os arquivos:
   - `index.html`
   - `style.css` 
   - `app.js`
   - `README.md`
3. Escreva uma mensagem de commit: "Site inicial Campos de Pinho"
4. Clique em "Commit changes"

### 3. Habilitar GitHub Pages
1. Vá em "Settings" do repositório
2. Role até "Pages" na barra lateral
3. Em "Source", selecione "Deploy from a branch"
4. Em "Branch", selecione "main"
5. Clique em "Save"

### 4. Acessar o Site
- O site estará disponível em: `https://seuusuario.github.io/campos-pinho-site/`
- Pode levar alguns minutos para ficar disponível

## 🔧 Personalizações

Para personalizar o site, edite:

- **Informações de contato**: Arquivo `app.js` → CONFIG
- **Conteúdo**: Arquivo `index.html`
- **Estilos**: Arquivo `style.css`
- **Cores**: CSS custom properties em `:root`

## 📱 WhatsApp Integration

O site possui 4 formas de contato via WhatsApp:

1. **Botão no Hero** - ao lado do título
2. **Formulário personalizado** - com nome e mensagem
3. **Botão direto** - na seção contato  
4. **Botão flutuante** - fixo no canto inferior
